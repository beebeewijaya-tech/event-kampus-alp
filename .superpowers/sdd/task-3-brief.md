### 2. RegistrationController

Create `app/Http/Controllers/RegistrationController.php`.

**create(Event $event)**:
- Load event categories: `$event->load('categories')`
- Get selected category id from query: `$selectedCategoryId = request('category')`
- Return view `registrations.create` with `$event`, `$categories = $event->categories`, `$selectedCategoryId`, `$user = auth()->user()`

**store(Event $event, StoreRegistrationRequest $request)**:
- Check if user already registered for this event (any category, any status):
  ```php
  $alreadyRegistered = Registration::whereHas('eventCategory', fn($q) => $q->where('event_id', $event->id))
      ->where('user_id', auth()->id())
      ->exists();
  if ($alreadyRegistered) {
      return back()->with('error', 'Anda sudah mendaftar untuk event ini.');
  }
  ```
- Get the category: `$category = EventCategory::findOrFail($request->event_category_id)`
- Check quota using DB lock:
  ```php
  $confirmedCount = Registration::where('event_categories_id', $category->id)
      ->where('status', 'confirmed')
      ->lockForUpdate()
      ->count();
  $status = $confirmedCount < $category->quota ? 'confirmed' : 'waiting_list';
  ```
  Wrap in `DB::transaction()`.
- Create registration:
  ```php
  $registration = Registration::create([
      'user_id' => auth()->id(),
      'event_categories_id' => $category->id,
      'status' => $status,
  ]);
  ```
  (check_in_code auto-generated by model boot)
- Create notification:
  ```php
  $message = $status === 'confirmed'
      ? 'Pendaftaran Anda untuk ' . $event->title . ' telah dikonfirmasi.'
      : 'Anda masuk dalam daftar tunggu untuk ' . $event->title . '.';
  Notification::create([
      'user_id' => auth()->id(),
      'registration_id' => $registration->id,
      'type' => 'confirm',
      'message' => $message,
  ]);
  ```
- Redirect: `return redirect()->route('registrations.index')->with('success', 'Pendaftaran berhasil!')`

**index()**:
- `$registrations = auth()->user()->registrations()->with('eventCategory.event')->latest()->get()`
- Return view `registrations.index` with `$registrations`
